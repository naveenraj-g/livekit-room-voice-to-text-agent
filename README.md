# LiveKit Transcription App

A minimal, production-ready LiveKit app where participants can join a room and see real-time transcripts generated by a Python agent.

## Prerequisites

- [LiveKit Server](https://docs.livekit.io/realtime/self-hosted/local-developer/) running locally or cloud.
- Node.js 18+
- Python 3.10+
- AssemblyAI API Key (or OpenAI).

## Setup

### 1. Backend (Python)

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your credentials
```

**Run:**
```bash
python main.py
```
Server runs on `http://localhost:8000`.

### 2. Frontend (Next.js)

```bash
cd frontend
npm install
cp .env.example .env.local
# Edit .env.local with your credentials
```

**Run:**
```bash
npm run dev
```
App runs on `http://localhost:3000`.

## Architecture

- **Next.js**: Handles UI, Room connection, and Token generation (`/api/token`).
- **Python Agent**: Connects to the room as a hidden participant, subscribes to audio, runs AssemblyAI STT, and pushes transcripts via SSE (`/transcript-stream`).

## API Endpoints

- `POST /api/token`: returns LiveKit token.
- `POST /api/start-transcriber`: triggers Python agent.
- `GET <backend>/transcript-stream?roomId=...`: SSE stream.
